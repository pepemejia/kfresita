# K-Fresita Vercel Ready

Este proyecto está listo para subirse directamente a GitHub y desplegarse en Vercel como sitio estático.

## Qué incluye
- Landing page completa
- Menú visual con tarjetas glass/liquid
- Pedido en línea con resumen de carrito
- Envío de pedido por WhatsApp
- Datos editables desde `data/config.json` y `data/site-content.json`

## Cómo usarlo
1. Sube todos los archivos a tu repositorio de GitHub.
2. Importa el repositorio en Vercel.
3. Vercel detectará un sitio estático.
4. Conecta `kfresita.com` y `www.kfresita.com` en Domains.

## Nota
Esta versión no incluye panel de administración persistente ni guardado de pedidos en archivos para que funcione correctamente en Vercel.
